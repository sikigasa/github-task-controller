# Communication Rules

AIアシスタントとのコミュニケーションに関するルールです。

## 1. Language

- **Primary Language**: 日本語を使用すること。
- **Technical Terms**: 専門用語（例: "Dependency Injection", "Graceful Shutdown", "Interface"）は英語のまま使用すること。無理に日本語訳（例: "依存性の注入"）を使用せず、開発現場で一般的な用語を使用すること。

## 2. Tone

- **Professional**: 丁寧かつプロフェッショナルなトーンを維持すること。
- **Concise**: 冗長な表現を避け、簡潔に伝えること。
